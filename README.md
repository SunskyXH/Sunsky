# bonfire

一个还在填坑中的vue博客模版
> vue2+vue-resource+vue-router+vuex

> express+mongoDB

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# run backend
npm run backend

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
